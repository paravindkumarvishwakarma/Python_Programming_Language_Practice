# Python_Programming_Language_Practice

A collection of small Python practice scripts and exercises grouped by course or topic. This repository is intended for learning, experimenting, and keeping short practice solutions organized.

**Repository Structure**

- `Daily_Practice/` — Short, day-by-day practice scripts (basic problems, patterns, and exercises).
- `Google_IT_Automation_With_Python/` — Notebooks and materials for the Google IT Automation with Python course.
- `Python_Mini_Module_Building/` — Small modules and utility scripts (examples and experiments).
- `Python_Practice_Apna_College/` — Practice exercises and notes from the Apna College course.
- `Python_Practice_Code_With_Harry/` — Files and notebooks from the Code With Harry tutorials.
- `SQL_Queries_Office/` — Useful SQL scripts for office tasks and maintenance.

**Highlights**

- Organized examples for beginners to intermediate Python learners.
- Mix of `.py` scripts and Jupyter notebooks for interactive learning.
- Small, focused files ideal for step-by-step practice and revision.

**Getting Started**

1. Install Python 3.8 or newer.
2. (Optional) Create and activate a virtual environment:

	python -m venv .venv
	.venv\\Scripts\\activate

3. Run a script:

	python Daily_Practice/D1_Hello_World.py

For notebooks, open them with Jupyter or VS Code's notebook support.
